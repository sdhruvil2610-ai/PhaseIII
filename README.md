# ⚙️ Aivena Workforce AI | Phase III: Executive Control Center

## Project Overview
Phase III represents the production-ready deployment of the **Aivena Workforce AI**. This project transitions Latin Leap's workforce management from a rigid, manual 48-hour "Peak-Coverage" scheduling model to an AI-driven, 40-hour compliant optimization engine. 

Designed for a retail network of 50 stores and 4,000+ employees, Aivena solves the structural conflict between the "Customer Wave" (dynamic foot traffic) and the "Human Brick" (rigid 8-hour shifts). The result is a mathematically proven **8.39% reduction in payroll costs** while maintaining 100% legal compliance and eliminating service gaps through strategic multi-role deployment.

---

## 🚀 Key Features

* **Sequential Optimization Pipeline:** Utilizes a highly stable, dictionary-indexed "Deep-Solve" engine that explores trillions of shift combinations to optimize 50 stores in under 10 minutes.
* **Authentic Legacy Comparison:** Features a "Shadow Simulator" that runs the exact same future customer traffic through historical human manager logic, proving that savings come from waste elimination, not blind cost-cutting.
* **Strike Force Analytics:** Isolates "Uncoverable Hours" into a single actionable KPI, prescribing the reinvestment of saved payroll into a fluid, multi-role workforce to protect peak revenue.
* **100% Legal Compliance:** Strict algorithmic enforcement of the 2027 Mexican 40-hour workweek mandate, zeroing out illegal overtime.

---

## 🛠️ Technical Architecture

This engine is built on **Prescriptive AI**, utilizing the Google OR-Tools CP-SAT solver. The objective function minimizes total payroll costs while aggressively penalizing service-level failures. 

To ensure maximum compatibility and ease of deployment for executive demonstrations, the system utilizes a **Flattened Data Architecture**. All seed data, scripts, and output files live in the root directory, preventing OS-level pathing errors.
